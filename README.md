# Mobilapplikasjon for Bon Appétit - IT2810 Prosjekt 4

Vi har valgt å utvikle en mobilapplikasjon som bygger videre på webapplikasjonen vi utviklet i prosjekt 3. Mobil applikasjonen gir mulighet til å søke blant hundrevis av oppskrifter. 

### På appen kan brukeren:
- Se oppskrifter i en liste
- Søke blant oppsrifter
- Filtrere på type
- Sortere på alfabetisk rekkefølge og på antall porsjoner fra flest til færrest.
